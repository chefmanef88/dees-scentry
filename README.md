# Dee's Scentry — Shop Luxury Perfumes

This package was generated from the uploaded Safari `.webarchive` of the Lovable preview.

## What is included
- `index.html` — the final saved storefront page, cleaned of Lovable preview badge/tracking.
- `assets/` — extracted local CSS, JavaScript bundles, and the hero perfume image available in the webarchive.

## How to open
Double-click `index.html`, or deploy the folder to any static host such as Vercel, Netlify, GitHub Pages, or cPanel.

## Important notes
- This is a static export from the public/saved preview, not the private editable Lovable source project.
- The uploaded page currently shows **“No perfumes yet”** in the catalog. Products appear to be managed from the Lovable/app dashboard and were not embedded in the saved page.
- Font files from the webarchive were intentionally not included. The page still references Google Fonts remotely.
